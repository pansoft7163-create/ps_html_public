📖 第一阶段：OCR 基础入门 (Day 1-3)
Day 1
入门
OCR 技术概述与发展历史
了解 OCR（光学字符识别）的基本概念、发展历程、应用场景及技术分类。掌握 OCR 技术的工作原理和基本流程。
OCR 基础概念 技术发展史 应用场景 工作流程

学习资源：

  • 什么是 OCR？传统 vs 深度学习 vs 大模型
  • OCR 的应用：票据识别、身份证识别、车牌识别等
  • OCR 技术演进：从模板匹配到深度学习
  • OCR 工作流程：图像预处理 → 文字检测 → 文字识别 → 后处理
Day 2
入门
传统 OCR 技术原理
学习传统 OCR 的核心技术，包括图像预处理、特征提取、模板匹配等方法。理解传统方法的优势和局限性。
图像预处理 二值化 特征提取 模板匹配

学习资源：

  • 图像预处理：去噪、二值化、倾斜校正
  • 特征提取方法：边缘检测、轮廓提取、HOG 特征
  • 传统识别算法：模板匹配、SVM 分类器
  • 传统方法的局限：对复杂场景适应性差

入门

Tesseract 安装配置 命令行使用 Python API

学习资源：

  • Tesseract 简介：Google 开源的 OCR 引擎
  • 安装 Tesseract 及中文语言包
  • 命令行基本用法和参数设置
  • Python pytesseract 库使用示例
  • 实战：识别简单的印刷体文字图像
🧠 第二阶段：深度学习 OCR 技术 (Day 4-7)
Day 4
进阶
深度学习基础与 CNN 应用
学习深度学习基础知识，重点掌握卷积神经网络（CNN）在图像特征提取中的应用原理。
深度学习基础 CNN 原理 特征提取 反向传播

学习资源：

  • 深度学习三要素：数据、算法、算力
  • CNN 基础：卷积层、池化层、全连接层
  • CNN 在 OCR 中的作用：自动学习图像特征
  • 常见 CNN 架构：LeNet、VGG、ResNet
Day 5
进阶
文字检测技术：CTPN、DBNet
深入学习文字检测算法，掌握如何从复杂图像中定位文字区域的技术方法。
文字检测 CTPN DBNet 边界框预测

学习资源：

  • 文字检测 vs 文字识别的区别
  • CTPN：基于 RNN 的文本检测方法
  • DBNet：可微分二值化网络
  • 检测算法评估指标：IoU、Precision、Recall
Day 6
高级
CRNN 文字识别算法详解
学习端到端文字识别的经典算法 CRNN，理解 CNN + RNN + CTC 的组合如何实现不定长文字识别。
CRNN LSTM CTC Loss 序列识别

学习资源：

  • CRNN 架构：CNN（特征提取）+ RNN（序列建模）+ CTC（解码）
  • LSTM/GRU：处理序列数据的循环神经网络
  • CTC Loss：解决序列对齐问题
  • CRNN 训练技巧和数据增强方法
Day 7
高级
Transformer 在 OCR 中的应用
了解最新的 Transformer 架构如何应用于 OCR 任务，学习 Attention 机制在文字识别中的优势。
Transformer Self-Attention TrOCR 大模型

学习资源：

  • Transformer 基础：Self-Attention 和 Multi-Head Attention
  • Vision Transformer (ViT) 在图像识别中的应用
  • TrOCR：基于 Transformer 的端到端 OCR 模型
  • 大模型 OCR：跨场景泛化和少样本学习能力
